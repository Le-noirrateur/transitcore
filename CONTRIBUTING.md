# Contributing

Thank you for considering contributing to TransitCore.

## Before contributing

Please discuss major changes before opening a Pull Request.

## Branches

Do not create Pull Requests against `main`.

Development Pull Requests must target:

- Neoforge
- Fabric
- Forge

depending on the affected loader.

## Coding Style

- Follow Java conventions.
- Document public APIs.
- Keep commits focused.
- Use meaningful commit messages.

## Pull Requests

Each Pull Request should:

- compile successfully;
- include a clear description;
- explain why the change is needed;
- avoid unrelated modifications.
