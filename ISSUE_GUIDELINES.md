# Bug / Crash Reports

Before opening an issue, please make sure that a similar report does not already exist. Duplicate reports will be closed to keep the tracker organized.

## How to report properly

To help the development team investigate your issue efficiently, please provide **at minimum**:

- **TransitCore version** (e.g. `0.1.0-alpha`)
- **Minecraft version** (e.g. `1.21.11`)
- **Loader and version** (NeoForge / Forge / Fabric)
- **Java version**
- **Operating System**
- **Other installed mods** (if applicable)
- **Description of the issue**: explain what you were doing, what happened, and what you expected to happen.
- **Complete logs** uploaded to **https://mclo.gs** or **https://pastebin.com**.
- **Crash report** (if the game crashed).
- **Steps to reproduce** the issue.

> **No logs = automatic issue closure.**

Do **not** upload screenshots of logs or paste raw logs directly into the issue.

# Feature Requests

Feature requests are welcome.

Before opening a feature request, please verify that it has not already been suggested.

A good feature request should include:

- A clear description of the proposed feature.
- The problem it solves.
- Why it would benefit TransitCore.
- Optional examples or references.

The development team reserves the right to reject suggestions that do not align with the project's vision.

---

# Proper Issue Usage

The issue tracker is intended for development purposes only.

The following behaviors are **not tolerated**:

- Spam.
- Duplicate reports.
- Off-topic discussions.
- Fake bug reports.
- Reports without required information.
- Abuse of the issue tracker.
- Repeatedly reopening closed issues.
- Attempting to modify labels through comments or requests.

Possible sanctions include:

- Warning.
- Temporary restriction from opening issues.
- Permanent ban from submitting issues.

Please keep discussions respectful, constructive, and professional.

# Labels

Issues are categorized exclusively by the development team.

## General

| Label | Description |
|-------|-------------|
| 🟠 **Bug** | Unexpected or incorrect behavior. |
| 🔴 **Crash** | TransitCore causes Minecraft to crash. |
| 🟣 **Incompatibility** | Conflict with another mod or loader. |
| 🔵 **On going** | A developer is currently working on the issue. |
| 🔵 **On hold** | Confirmed issue that will be fixed later. |
| 🟢 **Fixed** | The issue has been resolved. |
| 🟣 **Wontfix** | The issue will not be fixed. |
| ⚪ **Off-topic** | Not related to TransitCore. |
| ⚪ **Not-applicable** | Invalid or irrelevant report. |

---
